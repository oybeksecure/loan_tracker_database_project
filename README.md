# Loan Tracker Database System

## Overview
This project is a SQL Server database system for managing loans, borrowers, lenders, collateral, payments, and loan statuses.

The database provides efficient tracking of loan records and payment history while demonstrating advanced SQL Server features.

## Features
- Loan management
- Borrower and lender records
- Payment tracking
- Collateral management
- Loan status monitoring
- Payment summaries and balance calculations

## SQL Concepts Implemented
- Operators (Arithmetic, Comparison, Logical, Bitwise)
- Joins (Inner, Left, Right, Full, Self)
- Subqueries
- CASE Statements
- Date Functions
- Indexes
- Views
- Window Functions
- Common Table Expressions (CTE)
- Stored Procedures
- User Defined Functions
- Triggers

## Technologies
- Microsoft SQL Server
- T-SQL

## Learning Outcomes
This project strengthened practical knowledge of relational database design, query writing, reporting, automation using triggers and procedures, and SQL performance optimization.
